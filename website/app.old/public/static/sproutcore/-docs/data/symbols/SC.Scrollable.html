Namespace SC.Scrollable

Defined in: frameworks/sproutcore/mixins/scrollable.js

Any views you implement that are scrollable should include this mixin to provide basic support for scrolling actions. You can also override any of these methods as needed for your own specific behaviors.

Often times instead of adding SC.Scrollable to your view, you should place your view inside of an SC.ScrollView. See that class for more info.

Note that isScrollable must always be true.
