Class SC.Application

Extends: SC.Responder
Defined in: frameworks/sproutcore/foundation/application.js
Availability: since SproutCore 1.0

The SC.Application object manages a SproutCore application. A single instance is created and placed in a global variable named SC.app. All events and actions are routed through this object.

Property Details

keyPane

The pane that is currently receiving key events.

-   Object keyPane

Discussion

This is most often the SC.window object but if you display a popup or some other such element, you may set this to some other value.

mainPane

The main pane for the application.

-   Object mainPane

Discussion

The main pane is usually the primary pane you expect users to work in. It receives keyboard equivalent events and other operations even when another key pane is positioned over it.
